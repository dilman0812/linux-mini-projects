# Disk Usage Analyzer

## Description
This project analyzes disk usage on a Linux system and identifies directories consuming the most space.

## Commands Used
- df
- du
- sort
- awk
- sed

## How to Run
1. chmod +x disk_usage.sh
2. ./disk_usage.sh

## Output
- Disk usage report
- Top 10 largest directories
- Warning if usage exceeds threshold
